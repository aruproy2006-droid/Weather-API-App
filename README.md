# 🌦 Weather API App

A Python application that fetches live weather information using the OpenWeatherMap API.

---

## Features

- Live Weather
- Temperature
- Humidity
- Wind Speed
- Pressure
- Visibility
- Sunrise & Sunset
- Country Information
- Error Handling

---

## Technologies

- Python
- Requests
- JSON
- OpenWeatherMap API

---

## Installation

```bash
git clone https://github.com/yourusername/weather-api-app.git

cd weather-api-app

pip install -r requirements.txt
```

Run

```bash
python main.py
```
